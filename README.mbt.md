# gaojunran/mbt_algo

用 moonbit 语言练习算法。

每个源码文件夹中的 `GUIDE.md` 是由 LLM 生成的指导文档，使用如下的提示词生成：

```
我现在想学习手写 binary search tree，我希望你给我列出大概思路（需要实现什么，用什么实现）和注意事项，但不要教我具体如何实现。

输出到 `src/binary_search_tree/GUIDE.md` 中。
```
